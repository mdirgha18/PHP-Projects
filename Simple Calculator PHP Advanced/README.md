# Enhanced PHP Calculator

## Project Overview
This is a **web-based calculator** built using **PHP, HTML, CSS, and JavaScript**.  
It allows users to perform both **basic** and **advanced arithmetic operations**, with a clean and responsive UI.

---

## Features

- **Basic Operations:** Addition (+), Subtraction (-), Multiplication (×), Division (÷)  
- **Advanced Operations:** Power (^), Modulo (%), Square Root (√)  
- **Form Handling:** Retains user input after submission  
- **Input Validation:** Only numeric values allowed; division by zero handled  
- **User Experience Enhancements:**  
  - Clear button to reset the form  
  - Disable second number input for Square Root operation  
- **Responsive Styling:** Simple and professional layout for portfolio display  

---

## Technologies Used

| Technology | Purpose |
|------------|---------|
| PHP        | Handles form submission and server-side calculations |
| HTML       | Structures the calculator and user input form |
| CSS        | Styles the form, inputs, and results |
| JavaScript | Validates numeric input and manages the clear button |

---

## Installation / Usage

1. Create a **new PHP project** in your preferred environment (e.g., CodeSandbox, Replit, XAMPP/WAMP).  
2. Add a file named `index.php`.  
3. Copy the **Enhanced PHP Calculator code** into `index.php`.  
4. Run the project in the browser preview.  
5. Enter numbers, select an operation, and click **Calculate**.  
6. Use the **Clear** button to reset the form.
